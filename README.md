Animal Logic board game implemented in [Odin programming language](http://odin-lang.org) with Raylib

# Run
Executables are included. Clone this repo and inside the alogic-odin folder(directory),

for Windows, type
> alogic.exe

and for Linux, type
> ./alogic-odin.bin


# Build
> odin build .
